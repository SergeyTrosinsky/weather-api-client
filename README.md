# Weather Forecast Web Application

Веб-приложение для получения прогноза погоды через OpenWeather API.

Проект разработан на Django и позволяет пользователю получить актуальную информацию о погоде в выбранном городе через удобный веб-интерфейс. Данные загружаются из внешнего API и отображаются в браузере после обработки сервером.

В ходе работы были изучены основные компоненты Django: маршрутизация, представления, шаблоны и обработка пользовательских форм.

## Используемые технологии

- Python
- Django
- HTML
- OpenWeather API
- Requests

## Как работает приложение

Пользователь вводит название города в форму на веб-странице. Django отправляет запрос к OpenWeather API, получает данные о погоде и отображает результат в браузере.
