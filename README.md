# Weather API Client (Django)

## Описание проекта

Веб-приложение для получения актуального прогноза погоды через API сервиса OpenWeather.

Пользователь вводит название города в веб-форму, после чего приложение отправляет HTTP-запрос к внешнему API, получает информацию о текущих погодных условиях и отображает результат на странице.

---

## Функциональные возможности

- получение текущей погоды по названию города;
    
- интеграция с OpenWeather API;
    
- обработка HTTP-запросов;
    
- отображение температуры и погодных условий;
    
- работа с HTML-шаблонами Django;
    
- маршрутизация URL;
    
- обработка пользовательского ввода через формы.
    

---

## Используемые технологии

- Python
    
- Django
    
- OpenWeather API
    
- HTML
    
- CSS
    
- Requests

# Интеграция с OpenWeather API

Для получения данных используется сервис OpenWeather.

Приложение отправляет HTTP-запрос:

```python
requests.get(...)
```

и получает данные в формате JSON.

Из ответа извлекаются:

- температура;
    
- название города;
    
- описание погодных условий;
    
- дополнительные параметры погоды.
